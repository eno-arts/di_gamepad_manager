# DI Gamepad Manager

![Image](https://github.com/user-attachments/assets/a9c0781c-b835-4cc2-bec0-b275fcb3733a)

### description

Windowsが認識しているDirectInput(DI)ゲームパッドのデバイス状態を  
デバイスマネージャ同様の制御にて有効無効を切り替えることが出来ます。  
また有効なドライバのゲームパッドの状態を簡易的に確認することが出来ます。  

Windows 8からWindows10にて発生しているゲームパッドを繋いでいると  
PCがスリープしない問題でのゲームパッドの抜き差しの代用として作成しました。 
ゲームパッドをUSB接続したままドライバ無効で切り離すことが出来ます。

詳細は当方のブログ記事で確認して下さい。  
https://www.enoanoarts.com/2026/06/app-release-digamepad-manager.html  
Windowsのゲームパッドの有効無効を管理するアプリ「DI Gamepad Manager」を公開しました
